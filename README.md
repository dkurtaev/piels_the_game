# Piels

Browser game written on JavaScript.  
Gameplay of image opener arcade: character moves inside rectangle and draws
contour. When contour intersects with game field borders or with itself,
opening background image with corresponding shape. When enemy creatures
intersects contour during character's moving then game is ends.

This game stylized pie slicing: game field is a pie and character removes parts
of pie.  
Rendering style is an isometric pixel art.  
Game name formed like word 'Pixels' = 'Picture Elements' but 'Piels' = 'Pie
Elements'.
